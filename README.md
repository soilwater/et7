# kansas-eto-tables

A static website for looking up typical 7-day cumulative reference evapotranspiration (ET₀) by Kansas county. Select a county to view its reference table, see the typical ET₀ for the current week, and download the figure as JPG or PDF.

ET₀ values are derived from [GRIDMET](https://www.climatologylab.org/gridmet.html) climate data. Crop coefficient values are from FAO Irrigation and Drainage Paper No. 56 (Allen et al., 1998).
